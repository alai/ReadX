🧪 ReadX Chrome 插件测试页面

测试环境：本页面用于测试插件的核心功能，无需实际安装到Chrome。

📝 安装说明

  1. 打开 Chrome 浏览器，访问 chrome://extensions/
  2. 开启右上角的"开发者模式"
  3. 点击"加载已解压的扩展程序"
  4. 选择包含 ReadX 插件文件的文件夹
  5. 访问 X.com 的任意推文页面
  6. 点击工具栏中的 ReadX 图标

🎯 使用指南

第一版插件已完成！包含了所有要求的核心功能：
  • ✅ 智能内容提取（基于 data-testid 和语义化HTML）
  • ✅ 双击ESC或点击X按钮退出阅读模式
  • ✅ 字体、行距、宽度、字号自定义
  • ✅ Light/Dark 主题切换
  • ✅ 设置持久化保存
  • ✅ 响应式设计支持

⚠️ 注意事项

  • 图标文件: 需要创建实际的PNG图标文件替换 icons/ 目录下的占位符
  • 测试环境: 建议在实际的 X.com 页面上测试插件功能
  • 权限: 插件只请求必要的权限（activeTab, storage）
  • 兼容性: 支持 Chrome 88+ 和最新的 Manifest V3 标准
